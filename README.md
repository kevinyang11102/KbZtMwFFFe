# 前言

随着信息化时代的到来，智慧乡村旅游服务平台成为了推动乡村旅游发展的重要力量。本项目是基于Java语言和SSMPF（Spring、SpringMVC、MyBatis、PhpStudy/Navicat、Front-end）框架开发的乡村旅游服务平台，旨在为广大游客提供便捷、高效的乡村旅游服务。

## 内容介绍

本项目主要实现了以下功能模块：景区介绍、旅游攻略、在线预订、智慧导览等。通过使用Java语言和前端技术，实现了前后端分离的开发模式，使得项目具有较好的可维护性和可扩展性。此外，项目还结合了微信小程序，方便游客随时随地了解乡村旅游信息。

## 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16
